# PONG

Created using Unity 2019.4.17f1

**Paddle Controls**
- Default controls for Player One / Red Paddle:
  - Up    : **W**
  - Down  : **S**
  - Right : **D**
  - Left  : **A**
  
- Default controls for Player Two / Blue Paddle:
  - Up    : **Up Arrow Key**
  - Down  : **Down Arrow Key**
  - Right : **Left Arrow Key**
  - Left  : **Right Arrow Key**
